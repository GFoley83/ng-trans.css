Install

Use bower to install: bower install ng-trans-css, or download.


Usage

  1. Link your webpage to ng-trans.css
  2. Add class ng-trans ng-trans-{{transType}} to your the element of ng-repeat to make transition work.
  3. In addition, you can add class {{timingFunction}} to set timing fcuntion. (easeOutBack will bounce!)
              
      <ul class="list-group">
        <li class="list-group-item ng-trans ng-trans-{{transType}} {{timingFunction}}" ng-repeat="item in data">{{ item }}</li>
      </ul>
